Purpose: distributable custom OpenCode CLI enhancement stack for macOS that reproduces a curated local OpenCode setup on another machine with an additive installer. Tech stack: shell installer (install.sh), JSON config (opencode.json), TypeScript/npm package metadata, directories for skills/plugins/agents/commands/scripts/templates/hooks/rules/tools/platforms/vendor. Key behavior: install.sh is additive-only, copies missing files into ~/.config/opencode and ~/.local/bin, installs selected npm plugins, and intelligently merges opencode.json without overwriting existing user config/model choice.
